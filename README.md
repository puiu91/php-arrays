# php functions

A repo for commonly used functions and abstractions.

## Run Tests

```bash
git clone https://github.com/puiu91/functional-php-drills.git;
composer install;
cd php-functions;
clear; ./vendor/bin/phpunit --bootstrap vendor/autoload.php --testdox functions/tests;
```
